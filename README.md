## Install
server 와 client folder 각각
```sh
npm install
```

## Usage
1. server config folder 안에 prod.js를 참고
2. 해당 환경변수들을 local에서 쓸 수 있도록 js 생성
3. 그 후 아래 명령어 입력
```sh
npm run dev
```

## Run tests
